# projet-02 TEST-QI
1-- Description : création d'un Test d'intelligence en ligne, qui permet à un candidat de répondre à cinq questions en passant de page à l'autre, et de voir à la fin son score finale et une évaluation afficheés .  

2-- Technologies Utilisées : HTML, CSS, et Bases de PHP.

3-- Mise en marche : le projet sois stockée dans un serveur docker sur la machine local :
    3-1 ouvrir Docker Desktop.
    3-2 ouvrir un terminal dans l'emplacement de projet.
    3-3 activer les deux containteneur TestQI_php et TestQI_nginx via la commande Docker compose up -d.
    3-4 Sur un navigateur, taper sur la barre de recherche http://localhost:8080/.
